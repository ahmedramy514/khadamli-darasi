## 🎯 تدفق البيانات

### تسجيل حساب جديد:
```
المستخدم يملأ النموذج
        ↓
مصادقة البيانات في Frontend
        ↓
إرسال POST /api/auth/register
        ↓
Backend يتحقق من البيانات
        ↓
تشفير كلمة المرور (bcrypt)
        ↓
حفظ في MongoDB
        ↓
إرسال JWT Token
        ↓
حفظ في localStorage
        ↓
الانتقال للصفحة الرئيسية
```

### رفع سؤال جديد:
```
المستخدم يملأ النموذج
        ↓
التحقق من صحة البيانات
        ↓
إرسال POST /api/questions
        ↓
حفظ في MongoDB
        ↓
إرسال تنبيه (في المستقبل)
        ↓
عرض السؤال في الفصل
        ↓
اكتساب 10 نقاط (للإجابة)
```

## 🔗 تدفق الطلبات (Request Flow)

### مثال: جلب الفصول

```
Frontend                    Backend                   Database
   │                           │                          │
   ├─ GET /api/classrooms ─→   │                          │
   │                           ├─ Find classrooms ──→     │
   │                           │                          │
   │                  ←─ 200 OK ←─ [classrooms array]     │
   │                                                       │
   ├─ Render in React          │                          │
   │
```

### مثال: إضافة إجابة

```
Frontend                    Backend                   Database
   │                           │                          │
   ├─ POST /answer ───→        │                          │
   │   {text, questionId}      │                          │
   │                           ├─ Save answer ──→        │
   │                           │                          │
   │                           ├─ Add points to user     │
   │                           │  Update question ──→    │
   │                                                       │
   │                  ←─ 200 OK ←─ updated question      │
   │                                                       │
   ├─ Update UI                │                          │
   │
```

